Welcome to Scholarly

Scholarly is a powerful academic tool designed to assist faculty members in automating the process of summarizing their research publications. In an era where time and accuracy matter most, Scholarly reduces the manual effort of collecting, formatting, and organizing scholarly works by leveraging web scraping, natural language processing, and AI-driven summarization.

Whether it's for academic profiling, accreditation reports, or personal use, Scholarly ensures your research is presented professionally and efficiently — with just a few clicks.

How It Works

  1. Input: Faculty enters their Google Scholar profile link and email address.
  2. Scraping: Our system extracts relevant data such as titles, abstracts, authors, and citations using automated scraping tools.
  3. Summarization: Using NLP algorithms like PageRank and Cosine Similarity, each publication abstract is summarized. Incomplete abstracts are auto-completed using GPT-2.
  4. Presentation: The summarized data is displayed visually, downloadable as a PDF, and sent to the user’s email.

Core Features

  • Automated Web Scraping: Extracts data in real-time from Google Scholar.
  • AI-Based Summarization: NLP techniques ensure meaningful and concise summaries.
  • GPT-2 Completion: Fixes truncated abstracts with intelligent auto-completion.
  • PDF/DOCX Export: Download structured summaries for use in reports.
  • Email Integration: Get your summary sent directly to your inbox.
  • Visualizations: Year-wise publication charts generated using Matplotlib.
  • User-Friendly Interface: Accessible via a simple, interactive web app.

Technology Stack

The system is developed using the following modern tools and technologies:

  • Frontend: HTML, CSS, JavaScript, Streamlit (for UI)
  • Backend: Python with Flask (lightweight server-side logic)
  • Scraping: Selenium WebDriver, BeautifulSoup, Requests (data extraction)
  • NLP & AI: NLTK, NetworkX (PageRank), GPT-2 (HuggingFace Transformers)
  • Visualization: Matplotlib, Pandas
  • Reporting: ReportLab for PDF, python-docx for Word export
  • Email: smtplib for email delivery with SSL security

Who Can Use Scholarly?

This system can serve a wide range of users and institutions:

  • Faculty Members: Generate well-organized summaries for institutional profile updates.
  • Academic Institutions: Use for NAAC, NBA, and AICTE reports.
  • Research Scholars: Track and summarize research work effortlessly.
  • Department Coordinators: Maintain research stats and documentation year-wise.
  • Admin Staff: Reduce time spent on formatting faculty profiles manually.
